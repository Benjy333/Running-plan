# Website/App Starter

A clean starter you can deploy to **GitHub Pages** in minutes.

## Quick start
1. Edit `index.html`, `style.css`, and `script.js`.
2. Commit and push to GitHub.
3. Turn on **GitHub Pages**: Settings → Pages → Source = **GitHub Actions** (or `main` branch `/root`).
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Local preview
Use any static server (or just open `index.html` directly).
